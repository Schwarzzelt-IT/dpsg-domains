# DPSG Domains

Dieses Repository enthält eine Liste mit allen bekannten Domains von Gruppierungen der [Deutsche Pfadfinder*innenschaft Sankt Georg](https://dpsg.de) (DPSG).
Die Liste basiert auf den Daten der offiziellen "Liste der Gruppierungen - Anhang zu den Satzungen" mit Stand vom 13.01.2026.
Die Domains würden zwischen Dezember 2025 und Ende Januar 2026 recherchiert.

Die Liste erhebt keinen Anspruch auf Vollständigkeit oder Korrektheit.
Ergänzungen und Korrekturvorschläge sind jederzeit willkommen.
